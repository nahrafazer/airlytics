Airlytics adalah platform berbasis web yang dirancang untuk memantau parameter kualitas udara secara real-time dan mengklasifikasikannya menggunakan algoritma Machine Learning (Random Forest). Proyek ini merupakan bagian dari sistem pemantauan polusi udara yang mengintegrasikan perangkat keras, basis data, dan antarmuka pengguna yang interaktif.
